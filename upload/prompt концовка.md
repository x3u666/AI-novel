

# Экран концовки (Ending Screen) — Подробное описание

**URL:** `/ending` или отображается как overlay поверх игрового экрана

**Когда отображается:** Когда сюжет достигает финальной сцены (определяется флагом `isEnding: true` в mock-данных сцены). Переход к экрану концовки происходит **после** последнего сообщения рассказчика в чате — с задержкой 3–4 секунды, чтобы игрок успел прочитать финальные слова.

**Три типа концовок:**
| Тип | ID | Внутреннее название | Условие (на mock-данных) |
|---|---|---|---|
| Хорошая | `good` | Свет в конце пути | Набрано ≥ 70% «позитивных» решений (флаг `moralAlignment: 'good'`) |
| Нейтральная | `neutral` | Тень баланса | Смешанные решения, нет явного перевеса |
| Плохая | `bad` | Цена выбора | Набрано ≥ 70% «негативных» решений (флаг `moralAlignment: 'evil'`) |

---

## 1. Переход к экрану концовки (Transition)

### Последовательность

1. Последнее сообщение рассказчика в чате завершается (typewriter-эффект доиграл).
2. Пауза **3 секунды** — тишина, пользователь читает.
3. **Затемнение (Fade to Black):** Весь игровой экран (обе панели + toolbar) плавно затемняется (`opacity: 1 → 0`, duration: 1.5s, ease: easeInOut). Одновременно фоновая музыка (если была) плавно стихает (fade out, 2s).
4. **Чёрный экран** — 1.5 секунды полной темноты. Это создаёт драматическую паузу.
5. **Экран концовки** начинает проявляться (fade in, 2s).

### Звук при переходе

- Тихий, низкий, протяжный звук (drone/pad) начинает звучать во время чёрного экрана — свой для каждого типа концовки:
  - **Хорошая:** Тёплый, мажорный аккорд, медленно нарастающий.
  - **Нейтральная:** Нейтральный, медитативный тон — ни грустный, ни радостный.
  - **Плохая:** Низкий, диссонансный гул с лёгким реверсом.

## 3. Детальное описание каждого визуального элемента

### 3.1 Фон экрана концовки

Фон занимает весь viewport и фиксирован (`background-attachment: fixed`). Поверх фонового изображения/эффекта — полупрозрачный overlay для читабельности текста.

#### Хорошая концовка (`good`)

- **Фон:** Тёплый, светлый. Изображение рассвета/восхода солнца, залитого золотым светом пейзажа, или абстрактный градиент от глубокого тёмно-синего (сверху) к тёплому золотистому (снизу).
- **Overlay:** Радиальный градиент из центра — `rgba(212, 168, 83, 0.08)` (лёгкий золотистый оттенок).
- **Анимация фона:** Медленно перемещающиеся частицы-«светлячки» — маленькие точки (2–4px) золотистого цвета, плавно поднимающиеся вверх с лёгким покачиванием (CSS `@keyframes` или canvas). Количество: 15–25 частиц. Скорость: очень медленная (1 частица проходит экран за 15–25 секунд).
- **Vignette:** Мягкая, светлая — затемнение по краям минимальное, создающее ощущение простора.

#### Нейтральная концовка (`neutral`)

- **Фон:** Приглушённый, меланхоличный. Туманный пейзаж, дорога уходящая вдаль, или абстрактный градиент в серо-голубых тонах (от `#1A1A24` сверху к `#2A3040` снизу).
- **Overlay:** `rgba(168, 176, 188, 0.05)` — едва заметный серебристый.
- **Анимация фона:** Медленно плывущий туман — полупрозрачные белые/серые пятна (большие, размытые), горизонтально перемещающиеся слева направо. 3–5 слоёв с разной скоростью (параллакс-эффект). Или: медленно падающие листья / пылинки.
- **Vignette:** Умеренная — заметное затемнение по краям, фокус на центре. Ощущение задумчивости.

#### Плохая концовка (`bad`)

- **Фон:** Тёмный, тревожный. Грозовое небо, тлеющие угли, или абстрактный градиент от `#0A0A0E` (почти чёрный) к `#1A0A0A` (тёмно-бордовый оттенок).
- **Overlay:** `rgba(139, 46, 59, 0.06)` — едва уловимый кроваво-красный оттенок.
- **Анимация фона:** Медленно падающие частицы-«пепел» — маленькие точки (1–3px) тёмно-серого / тускло-оранжевого цвета, медленно падающие вниз с лёгким вращением. Количество: 20–30 частиц. Или: медленные вспышки далёких «молний» — кратковременное (0.1s) осветление фона на 5–10%, раз в 8–12 секунд.
- **Vignette:** Сильная — глубокое затемнение по краям, экран как бы сужается, давит. Ощущение безысходности.

### 3.2 Декоративный символ

Маленький символ над словом «КОНЕЦ» — задаёт тон.

| Концовка | Символ | Описание |
|---|---|---|
| Хорошая | `✦` или `☀` | Звезда / солнце, цвет `#D4A853` (золотой), лёгкое свечение (box-shadow) |
| Нейтральная | `◈` или `∞` | Ромб / бесконечность, цвет `#A8B0BC` (серебристый), без свечения |
| Плохая | `✧` или `☽` | Потухшая звезда / полумесяц, цвет `#8B2E3B` (бордовый), тусклое мерцание |

- **Размер:** 24–32px.
- **Анимация появления:** Fade-in, задержка 1s после начала fade-in экрана. Для хорошей — лёгкая пульсация свечения (scale 1.0 → 1.1, бесконечно, медленно). Для плохой — медленное мерцание (opacity 0.6 → 1.0, бесконечно).
- **Отступ:** 40px от слова «КОНЕЦ» сверху.

---

### 3.3 Надпись «КОНЕЦ»

- **Шрифт:** Playfair Display, Bold, 40–48px.
- **Выравнивание:** По центру горизонтально.
- **letter-spacing:** 8–12px (разрядка для торжественности).
- **text-transform:** uppercase.

**Цвет — зависит от типа:**

| Концовка | Цвет текста | Дополнительный эффект |
|---|---|---|
| Хорошая | `#D4A853` (золотой) | Лёгкий text-shadow: `0 0 20px rgba(212, 168, 83, 0.3)` — золотистое свечение |
| Нейтральная | `#A8B0BC` (серебристый) | Без свечения, или едва заметный `0 0 10px rgba(168, 176, 188, 0.15)` |
| Плохая | `#8B2E3B` (бордовый) | text-shadow: `0 0 15px rgba(139, 46, 59, 0.25)` — тёмно-красное тусклое свечение |

**Анимация появления:**
1. Буквы появляются **по одной** (stagger, 0.08s на букву) с эффектом fade-in + scale (от 0.8 к 1.0). Общее время: ~0.5s для слова «КОНЕЦ».
2. Задержка: 2.5s после начала fade-in экрана.
3. После полного появления — задержка 1s, затем появляется название концовки.

### 3.4 Название концовки и подзаголовок

**Название концовки:**
- **Шрифт:** Playfair Display, Italic, 22–26px.
- **Цвет:** Тот же, что у «КОНЕЦ», но с opacity 0.85.
- **Обрамление:** Кавычки-ёлочки: `«Свет в конце пути»`.
- **Анимация:** Fade-in + slide-up (y: 10px → 0), duration: 0.6s. Задержка: 3.5s.

**Названия для каждого типа:**

| Концовка | Название | Подзаголовок-эпитет |
|---|---|---|
| Хорошая | «Свет в конце пути» | Вы нашли свой путь |
| Нейтральная | «Тень баланса» | Не всё потеряно. Не всё найдено. |
| Плохая | «Цена выбора» | Каждое решение имело последствия |

**Подзаголовок:**
- **Шрифт:** Inter, Regular, 14–16px.
- **Цвет:** Вторичный текст (`#9898A6`).
- **Анимация:** Fade-in, задержка: 4.2s. Duration: 0.5s.
- **Отступ:** 8px ниже названия.

### 3.5 Финальный текст (Epilogue)

Основной нарративный блок, подводящий итог истории. **Персонализируется** под выборы игрока на протяжении игры.

**Контейнер:**
- **Max-width:** 620px, центрирован.
- **Фон:** Полупрозрачный — `rgba(15, 15, 20, 0.6)` с backdrop-blur (`backdrop-filter: blur(8px)`). Это обеспечивает читабельность на любом фоне.
- **Рамка:** `1px solid rgba(255, 255, 255, 0.06)`.
- **Border-radius:** 12px.
- **Padding:** 28px 32px.
- **Отступ сверху:** 32px от подзаголовка.

**Текст:**
- **Шрифт:** Playfair Display, Regular, 17px. Line-height: 1.8.
- **Цвет:** Основной текст (`#E8E8ED`).
- **Длина:** 2–4 абзаца (150–300 слов).
- **Анимация:** Typewriter-эффект (посимвольное появление), скорость из настроек. Задержка: 5s. Это главный «момент» — игрок читает финал как разворачивающийся текст.
- **Между абзацами:** Отступ 16px.

**Содержание финального текста — отличается для каждого типа:**

#### Хорошая концовка — текст

Тон: **тёплый, оптимистичный, завершённый.** Чувство, что усилия вознаграждены. Упоминаются конкретные решения, которые привели к хорошему исходу.

*Пример (пресет «Рыцарь»):*

> Стены крепости, ещё недавно объятые тьмой, вновь сияли в лучах рассвета. Знамёна, изорванные в битвах, были подняты заново — не как символ войны, но как свидетельство того, что свет не может угаснуть, пока есть те, кто готов за него стоять.
>
> Вы сделали выбор, который многие сочли бы безрассудным, — протянули руку врагу вместо меча. И именно это решение изменило всё. Элрон, старый мудрец, которого вы не бросили в подземельях, оказался ключом к примирению — его слова разрушили проклятие древнее самих стен.
>
> Мир наступил. Не потому, что зло было уничтожено, а потому, что кто-то решил понять его. Ваш путь стал легендой — не о великих битвах, а о великом сострадании.

#### Нейтральная концовка — текст

Тон: **задумчивый, двойственный, философский.** Чувство незавершённости — не всё хорошо, не всё плохо. Вопросы остаются без ответа.

*Пример (пресет «Рыцарь»):*

> Дорога уходила вдаль, и вы шли по ней один. Позади остались и победы, и потери — Элрон ушёл своим путём, Лиана так и не простила последнего решения, а крепость стояла — не разрушенная, но и не восстановленная.
>
> Вы сделали, что могли. Или не всё? Этот вопрос будет преследовать вас ещё долго — в тех тихих вечерах, когда ветер напоминает звук далёких битв, а тени ложатся так, что кажется: кто-то стоит за спиной.
>
> История не закончилась. Она просто перестала рассказывать себя вслух. Возможно, в другой раз — с другим выбором — она зазвучит иначе.

#### Плохая концовка — текст

Тон: **мрачный, тяжёлый, предостерегающий.** Чувство утраты и последствий. Конкретные решения, которые привели к краху, упоминаются прямо.

*Пример (пресет «Рыцарь»):*

> Пепел — вот всё, что осталось. Пепел и тишина.
>
> Крепость пала не от силы врага — она пала изнутри. Когда вы отвернулись от Элрона в подземельях, вы потеряли не просто союзника. Вы потеряли единственного, кто знал слово, снимающее проклятие. Когда подняли меч вместо того, чтобы протянуть руку, — вы получили победу в битве и проигрыш в войне.
>
> Теперь враг повержен, но и победитель стоит среди руин. Знамёна сгорели. Никто не поднимет их заново — некому поднимать и незачем. Каждый ваш шаг был выбором. И каждый выбор имел цену. Оглянитесь — и посчитайте, сколько заплачено.

- **Анимация:** Fade-in, задержка: после завершения typewriter-эффекта финального текста + 1s. Duration: 0.6s.

### 3.7 Блок статистики

**Контейнер:**
- Аналогичен контейнеру финального текста (полупрозрачный фон с blur), но чуть менее прозрачный.
- **Max-width:** 480px.
- **Padding:** 24px 28px.
- **Border-radius:** 12px.
- **Отступ сверху:** 24px от разделителя.

**Строки статистики:**

Каждая строка — горизонтальный ряд:
```
[Иконка] [Метка]                    [Значение]
```

| Иконка | Метка | Пример значения | Описание |
|---|---|---|---|
| ⏱ | Время прохождения | `01:12:35` | Общее игровое время (`GameState.totalPlayTimeSeconds`, форматированное) |
| 🔀 | Принято решений | `14` | Количество элементов в `GameState.decisions` |
| 👥 | Встречено персонажей | `8` | Количество элементов в `GameState.characters` |
| 📖 | Глав пройдено | `5` | Максимальный `chapterNumber` в `narrativeHistory` |
| ⚖️ | Ваш путь | *Путь героя* / *Путь тени* / *Путь странника* | Зависит от `moralAlignment` |
| 🏆 | Концовка | `1 из 3` / `2 из 3` / `3 из 3` | Номер текущей + общее количество |

**Стиль строк:**
- **Иконка:** 18px, слева.
- **Метка:** Inter, Regular, 14px, вторичный цвет. После иконки, отступ 8px.
- **Значение:** Inter, SemiBold, 14px, основной цвет. Выравнено вправо (`justify-between`).
- **Между строками:** 12px.
- **Строка «Ваш путь»** — значение окрашено в цвет, зависящий от типа:

| Концовка | Текст «Путь» | Цвет значения |
|---|---|---|
| Хорошая | Путь героя | `#5CAA7E` (зелёный) |
| Нейтральная | Путь странника | `#A8B0BC` (серебристый) |
| Плохая | Путь тени | `#C45C5C` (красный) |

**Анимация появления:**
- Весь блок — fade-in + slide-up. Задержка: после разделителя + 0.5s.
- Каждая строка — stagger, 0.1s. Значения появляются с эффектом «счётчика» (числа быстро увеличиваются от 0 до конечного значения за 0.8s — `count-up animation`). Время форматируется без анимации (появляется сразу).

### 3.8 Финальная цитата рассказчика

Короткая фраза «от лица» рассказчика — прощание с игроком. Зависит от пресета и типа концовки.

**Контейнер:**
- Без фона (прозрачный) или с минимальным `rgba(255,255,255,0.03)`.
- Вертикальная декоративная линия слева (3px, акцентный цвет пресета, opacity 0.4).
- **Padding-left:** 20px.
- **Max-width:** 520px, центрирован.
- **Отступ сверху:** 28px от статистики.

**Текст:**
- **Шрифт:** Playfair Display, Italic, 16px.
- **Цвет:** Основной текст, opacity 0.8.
- **Содержание:** 1–2 предложения.

**Примеры (пресет «Рыцарь»):**

| Концовка | Цитата |
|---|---|
| Хорошая | *«Каждый подвиг начинается с одного шага, сделанного вопреки страху. Ты сделал этот шаг — и мир стал светлее. Прощай, герой. До новых сказаний.»* |
| Нейтральная | *«Не каждая история заканчивается триумфом или трагедией. Иногда она просто заканчивается. Но может начаться снова — с другим выбором.»* |
| Плохая | *«Я рассказал историю, которую ты написал своими решениями. Не вини рассказчика за финал — каждая строка была твоей. Но если хочешь — перепиши.»* |

**Подпись:**
- Под цитатой, справа: `— Рыцарь, ваш рассказчик` (Inter, Regular, 12px, вторичный цвет, курсив).

**Анимация:** Fade-in, задержка: после статистики + 1s. Duration: 0.8s.

### 3.9 Кнопки действий

**Расположение:** Под цитатой, вертикальная колонка, центрированы.

7.0s  ─── Финальный текст: начало typewriter-эффекта
         (длительность зависит от скорости в настройках,

~23s  ─── Typewriter завершён

## 8. Определение типа концовки (логика)

На текущем этапе (без LLM) тип концовки определяется **подсчётом решений** с моральными тегами, заложенными в mock-данных:

```typescript
// В MockChoice добавляется поле:
interface MockChoice {
  id: string;
  text: string;
  nextSceneId: string;
  shortLabel: string;
  moralWeight: number;  // -1 (негативное), 0 (нейтральное), +1 (позитивное)
}

// Логика определения концовки:
function determineEndingType(decisions: Decision[], mockData: MockData): 'good' | 'neutral' | 'bad' {
  const totalWeight = decisions.reduce((sum, d) => {
    const choice = findChoiceById(d.choiceId, mockData);
    return sum + (choice?.moralWeight ?? 0);
  }, 0);
  
  const maxPossible = decisions.length; // если все +1
  const ratio = totalWeight / maxPossible;
  
  if (ratio >= 0.4) return 'good';       // Перевес к позитивным
  if (ratio <= -0.4) return 'bad';       // Перевес к негативным
  return 'neutral';                       // Смешанные решения
}
```

Эта логика может быть вынесена в `services/narrativeService.ts` для будущей замены на LLM-определение.
