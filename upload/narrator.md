

# Экран выбора нарратора (Narrator Selection) — Подробное описание дизайна

**URL:** `/select-narrator`

**Предусловие:** Пользователь нажал «Новая игра» в главном меню.

**Цель экрана:** Дать пользователю выбрать стилистический пресет (персонажа-рассказчика), который определит тон, жанр и эстетику всей будущей истории. Экран должен быть **витриной** — каждый пресет должен вызывать эмоцию и желание попробовать. Это момент предвкушения перед началом истории.

---

## 1. Общая концепция дизайна

**Метафора:** Экран выбора — это «зал рассказчиков», «библиотека историй» или «галерея масок». Пользователь проходит мимо рассказчиков, каждый из которых приглашает его в свой мир. Каждый рассказчик — это не просто кнопка, а **погружение**: при фокусе на рассказчике весь экран адаптируется под его эстетику.

**Ключевая идея: «Экран дышит пресетом».** Когда пользователь наводит/выбирает рассказчика, фон, акцентные цвета, атмосферные частицы и даже тонкие звуковые элементы меняются, давая предварительное ощущение стиля. Это создаёт эмоциональную привязку к выбору ещё до начала игры.

---

## 2. Переход с главного меню

### Анимация входа

1. Главное меню: fade-out (0.4s).
2. Промежуточное состояние: тёмный экран (0.2s).
3. Экран выбора: fade-in фона (0.5s) → toolbar slide-down сверху (0.3s) → карточки пресетов появляются stagger-анимацией (описано ниже) → область описания fade-in (0.3s) → кнопка «Продолжить» fade-in (0.2s).

Общее время входа: ~1.5s.

## 4. Фон экрана

### Основной фон

Фон **не статичный** — он реагирует на выбранного рассказчика.

**Базовое состояние (ничего не выбрано / «Нейтральный»):**
- Основной фон приложения (`#0F0F14`).
- Лёгкий радиальный градиент из центра — чуть светлее в центре (`rgba(255,255,255,0.02)`), создающий ощущение «сцены».
- Никаких частиц или анимаций.

**При выборе / hover на конкретном пресете** — фон плавно (transition: 0.8s ease) трансформируется:

| Пресет | Изменение фона | Частицы / эффекты |
|---|---|---|
| Нейтральный | Без изменений, базовый | Нет |
| Рыцарь | Радиальный градиент с `rgba(212, 168, 83, 0.04)` из нижней части | Едва заметные «искры» — 5–8 золотых точек (1–2px), медленно поднимающиеся от низа |
| Романтик | Радиальный градиент с `rgba(212, 123, 142, 0.04)` из центра | Медленно падающие лепестки — 3–5 полупрозрачных розовых форм, вращающихся при падении |
| Боец | Радиальный градиент с `rgba(212, 91, 62, 0.04)` из нижнего правого угла | Мелкие «тлеющие угольки» — 5–7 красно-оранжевых точек, хаотично движущихся |
| Учёный | Радиальный градиент с `rgba(75, 163, 212, 0.04)` из верхней части | Тонкие горизонтальные линии-«сканеры» — 2–3 полупрозрачные голубые линии, медленно ползущие сверху вниз |
| Тёмный маг | Радиальный градиент с `rgba(139, 92, 212, 0.04)` из центра | «Руны» — 4–6 мелких фиолетовых символов (∞, ◈, △), медленно вращающихся на фиксированных позициях, opacity 0.06 |
| Детектив | Радиальный градиент с `rgba(201, 168, 76, 0.03)` из верхнего левого угла | Эффект «дождя за окном» — диагональные тонкие линии, едва различимые, opacity 0.03 |
| Хоррор | Vignette усиливается, углы экрана становятся темнее | «Мерцание» — фон слабо и нерегулярно пульсирует (opacity 0.02 → 0.05 → 0.02), раз в 5–8 секунд |
| Поэт | Радиальный градиент с `rgba(123, 168, 150, 0.03)` из верхней части | «Листья» — 3–4 силуэта листьев, медленно падающих по диагонали, opacity 0.04 |
| Комедиант | Радиальный градиент с `rgba(232, 168, 56, 0.04)` из центра | «Конфетти» — 6–8 мелких разноцветных точек, медленно опускающихся, с лёгким покачиванием |

**Важно:** Все фоновые эффекты **крайне тонкие** (opacity 0.03–0.06). Они не должны отвлекать — только создавать подсознательное ощущение атмосферы. Если пользователь не обратит на них внимания — это нормально. Но если обратит — будет приятное «ого, экран меняется».

**Переход между фонами:** CSS transition на `background`, duration: 0.8s, ease: easeInOut. Частицы/эффекты: fade-in/fade-out duration: 0.5s. При смене пресета — старые частицы fade-out, новые fade-in (без наложения).

## 5. Toolbar

Стандартный toolbar проекта (описан в основном гайде). На этом экране:

**Высота:** 60px.
**Фон:** `#1A1A24` с `border-bottom: 1px solid rgba(255, 255, 255, 0.06)`.

**Элементы слева:**
- **Логотип / название проекта:** Playfair Display, Bold, 20px, цвет `#E8E8ED`. При hover — цвет акцентный. Кликабелен → возврат в главное меню (без подтверждения, т.к. игра ещё не началась).

**Элементы по центру:**
- Пусто (или тонкий текст «Новая игра», Inter, Regular, 12px, вторичный цвет, opacity 0.5 — для контекста).

**Элементы справа:**
- **⚙️ Настройки** — модалка настроек. Hover: tooltip «Настройки», цвет иконки `#9898A6` → `#E8E8ED`.
- **🏠 Главное меню** — возврат в меню. Hover: tooltip «Главное меню».
- Иконки 💾 (Сохранить) и 📖 (Дневник) **отсутствуют** на этом экране (игра ещё не началась).

**Тонкая акцентная полоска:** Под toolbar — горизонтальная линия высотой 2px, цвет — акцентный цвет текущего выбранного пресета. Transition: 0.5s ease. По умолчанию — `#A8B0BC` (нейтральный). При смене пресета плавно меняет цвет. Это первый визуальный сигнал, что экран «чувствует» выбор.

## 7. Карусель карточек пресетов

### Общая структура

Горизонтальная полоса с карточками, расположенная в центре экрана. Это **не сетка**, а **линейная карусель**: карточки расположены в один ряд, при необходимости можно прокрутить горизонтально.

**Контейнер карусели:**
- **Ширина:** 100% экрана, max-width: 1100px, центрирован.
- **Отступ сверху:** 28px от подзаголовка.
- **Высота контейнера:** 190px (карточки + отступы + индикаторы).
- **Overflow:** Hidden (прокрутка управляется стрелками, свайпом или клавишами).
- **Фон контейнера:** Прозрачный.

### Расположение карточек

**На десктопе (≥ 1280px):** Видимы **5 карточек** одновременно. Если пресетов больше (10) — остальные доступны прокруткой.

**Распределение в ряду:**
```
  ◀   [ Карт.1 ] [ Карт.2 ] [ Карт.3★ ] [ Карт.4 ] [ Карт.5 ]   ▶
                                 ↑
                         активная (по центру)
```

**Центрирование активной карточки:** Выбранная/активная карточка всегда стремится к центру ряда. При переключении — остальные карточки плавно сдвигаются (translateX), чтобы новая активная оказалась по центру.

### Кнопки прокрутки (стрелки)

- **Расположение:** По бокам карусели, вертикально центрированы.
- **Вид:** Круглые кнопки, 36×36px, фон `rgba(255, 255, 255, 0.06)`, border: `1px solid rgba(255, 255, 255, 0.1)`. Иконка — шеврон `‹` / `›`, цвет вторичного текста.
- **Hover:** Фон `rgba(255, 255, 255, 0.12)`, иконка — основной цвет.
- **Поведение:** Сдвиг карусели на 1 карточку. Анимация: translateX, duration: 0.35s, ease: easeOut.
- **Неактивная стрелка:** Если достигнут край — opacity 0.3, cursor: default, клик игнорируется.
- **Скрытие:** Если все карточки помещаются без прокрутки — стрелки скрыты.

### Точки-индикаторы

Под карточками — ряд точек, показывающих текущую позицию в карусели.

```
●  ●  ●  ○  ○
```

- **Количество точек** = количество «страниц» карусели (или = количеству пресетов, если прокрутка поштучная).
- **Активная точка:** Акцентный цвет текущего пресета, размер 8px.
- **Неактивная:** `rgba(255, 255, 255, 0.2)`, размер 6px.
- **Отступ сверху:** 16px от нижнего края карточек.
- **Переход:** Плавное изменение цвета и размера (transition: 0.3s).

### Клавиатурная навигация

- `←` / `→` — переключение между карточками.
- `Enter` — подтверждение выбора (аналог клика по карточке).
- **Focus ring:** При навигации клавиатурой — у активной карточки появляется outline акцентного цвета (2px, offset 3px).

### Свайп (тач-устройства)

- Горизонтальный свайп по области карусели — прокрутка.
- Используется `touch-action: pan-x`.
- Инерция: после свайпа — карточки «доезжают» до ближайшей позиции (snap).

### Состояния карточки

#### Default (не выбрана, не hover)

```css
background: #1E1E2A;
border: 1px solid rgba(255, 255, 255, 0.06);
transform: scale(1);
box-shadow: none;
opacity: 0.75;
```

Карточка слегка приглушена — это визуально «уступает место» активной карточке.

#### Hover (курсор наведён, но не выбрана)

```css
background: #242434;
border: 1px solid rgba(255, 255, 255, 0.12);
transform: scale(1.04) translateY(-4px);
box-shadow: 0 8px 24px rgba(0, 0, 0, 0.3);
opacity: 0.9;
transition: all 0.25s ease;
```

- Карточка «приподнимается» над рядом.
- Фон чуть светлее.
- Тень усиливается, создавая ощущение объёма.
- Аватар масштабируется (scale 1.08).

#### Selected (активная, выбранная)

```css
background: #282840;
border: 2px solid [акцентный цвет, opacity 0.6];
transform: scale(1.08) translateY(-6px);
box-shadow: 
  0 8px 32px rgba(0, 0, 0, 0.4),
  0 0 20px [акцентный цвет, opacity 0.1];
opacity: 1;
transition: all 0.3s ease;
```

- Карточка **заметно выделяется** среди остальных.
- Рамка — акцентный цвет пресета.
- Лёгкое свечение (glow) вокруг карточки цветом пресета.
- Полная непрозрачность.
- Акцентная полоска внизу — полная непрозрачность, расширена.
- **Маркер выбора:** Маленький чекмарк (✓) в правом верхнем углу карточки, внутри круга (18px) с фоном акцентного цвета. Fade-in + scale (0.3s) при выборе.

```
      ┌────────────────────┐
      │              [✓]   │  ← чекмарк, 18px круг, акцентный фон
      │    ┌──────────┐    │
      │    │  Аватар  │    │
      ...
```

#### Соседние карточки (рядом с активной)

Карточки, стоящие непосредственно рядом с активной (±1 позиция), имеют opacity 0.85 (чуть ярче, чем дальние). Это создаёт градиент внимания от центра к краям.

Карточки на ±2 и дальше: opacity 0.65.

### Анимация появления карточек при входе на экран

**Stagger-анимация:** Каждая карточка появляется с задержкой.

```
Карточка 1: задержка 0.8s, fade-in + slide-up (y: 30px → 0), scale (0.9 → 1), duration: 0.4s
Карточка 2: задержка 0.9s, то же
Карточка 3: задержка 1.0s, то же
Карточка 4: задержка 1.1s, то же

```

Stagger: 0.1s. Карточки «всплывают» снизу, одна за другой. Эффект: как будто рассказчики «выходят на сцену».

**После появления всех карточек:**
- Карточка «Нейтральный рассказчик» (или первая) автоматически получает состояние Selected (с задержкой 0.3s после последней карточки).

### 12.1 Звуковые эффекты (если звук включён в настройках)

| Действие | Звук | Описание |

### 12.2 Состояния при первом входе

**Если это самый первый визит пользователя (нет данных в localStorage):**
- После появления карточек — над «Нейтральным рассказчиком» на 4 секунды появляется тултип-подсказка:
  ```
  ┌────────────────────────────────────┐
  │ 💡 Нажмите на рассказчика,         │
  │    чтобы узнать больше о его стиле │
  └────────────────────────────────────┘
                    ▼
  ```
  - Фон: `rgba(255, 255, 255, 0.1)`. Border-radius: 8px. Inter, Regular, 13px.
  - Fade-in (0.3s), задержка 2s после появления карточек.
  - Fade-out через 4s или при любом взаимодействии с карточкой.
  - Показывается **только один раз** (сохраняется флаг `hasSeenNarratorHint: true` в localStorage).

### 12.3 Переход к следующему экрану (подробная анимация)

При клике «Продолжить»:

1. **0ms:** Кнопка переходит в состояние «загрузка» (текст → «Начинаем...», лёгкий spinner).
2. **200ms:** Все карточки, кроме выбранной, плавно fade-out (opacity → 0, duration: 0.3s).
3. **400ms:** Выбранная карточка — плавно перемещается к центру экрана и увеличивается (scale: 1.08 → 1.5, translateY к центру viewport). Duration: 0.5s.
4. **700ms:** Область описания и кнопка — fade-out (0.2s).
5. **900ms:** Карточка продолжает увеличиваться (scale → 3.0, opacity → 0 — «растворяется»). Одновременно весь экран fade-to-white (или fade-to-accent-color — лёгкая вспышка цвета пресета). Duration: 0.4s.
6. **1300ms:** Экран полностью перекрыт. Начинается загрузка Экрана 3.
7. **1500ms:** Fade-in Экрана 3.

**Упрощённый вариант (если сложная анимация избыточна):**
1. Кнопка → «Начинаем...».
2. Весь экран — fade-out (0.5s).
3. Чёрный экран — 0.3s.
4. Fade-in Экрана 3.

### 13.1 Нейтральный рассказчик

- **Подзаголовок:** Универсальная история
- **Описание:**

  >
  > Язык повествования спокойный и ровный, без стилистических крайностей. Средний уровень детализации, сбалансированный темп. Идеальный выбор, если вы хотите создать историю в свободном формате и сами определить её направление.

- **Теги:** `#универсальный` `#свободный стиль` `#классика` `#любой жанр`
- **Пример:** *«Дорога вилась между холмами, теряясь в утренней дымке. Вы остановились на развилке. Направо уходила узкая тропа в лес, налево — широкий тракт к далёкому городу. Где-то вдалеке раздался крик птицы.»*

### 13.2 Рыцарь

- **Подзаголовок:** Приключения
- **Описание:**

  > Средневековая эстетика с эпическим размахом. Древние замки, заколдованные леса, мифические существа и забытые клятвы. Вы — герой, отправляющийся в опасное путешествие, где каждый выбор определяет судьбу королевств.
  >
  > Язык возвышенный и торжественный, с архаичными оборотами. Длинные описательные предложения, богатые метафоры. Рассказчик ведёт историю так, будто записывает её на пергамент при свете свечи.
  >
  > Ожидайте выборы между честью и выгодой, схватки с чудовищами и моменты, когда судьба целого мира зависит от одного решения.

- **Теги:** `#средневековье` `#эпос` `#герой` `#мифы` `#рыцарство` `#судьба`
- **Пример:** *«И встал рыцарь пред вратами, что были высечены в скале самими древними. Ветер нёс шёпот забытых клятв. Меч в его руке пел, предвкушая битву, ибо за этими вратами ждал враг, чьё имя не произносили вслух уже тысячу лет.»*

### 13.3 Романтик

- **Подзаголовок:** Любовная история
- **Описание:**

  > Чувственное повествование, где главная сила — не меч и не магия, а человеческое сердце. Сложные отношения, любовные треугольники, моменты нежности и боли. Каждый диалог — признание, каждый взгляд — история.
  >
  > Язык лиричный и эмоциональный. Много описаний чувств, ощущений, мимолётных жестов, значащих больше слов. Внутренние монологи, поэтичные метафоры.
  >
  > Ваши выборы будут решать не исход битвы, а судьбу отношений — кого полюбить, кому довериться, и чьё сердце разбить.

- **Теги:** `#романтика` `#чувства` `#отношения` `#драма` `#поэзия` `#сердце`
- **Пример:** *«Она отвернулась, но он заметил, как дрогнули её пальцы, сжимавшие край письма. В этом движении было всё — и страх, и надежда, и та тихая, непроизнесённая просьба, которую он не мог не услышать. Дождь за окном стучал по крыше, словно считая секунды до слов, которые изменят всё.»*

### 13.4 Боец

- **Подзаголовок:** Экшен
- **Описание:**

  > Адреналин, скорость, опасность на каждом шагу. Короткие фразы, рубленый ритм. Здесь не рассуждают — здесь действуют. Бои, погони, взрывы, выживание против всех шансов.
  >
  > Минимум рефлексии — максимум динамики. Рассказчик говорит так, будто каждое слово — пуля. Описания телесные, физические: удары, движения, звуки.
  >
  > Решения принимаются за секунды. Ошибка — и всё кончено. Готовы?

- **Теги:** `#экшен` `#бой` `#динамика` `#выживание` `#адреналин` `#скорость`
- **Пример:** *«Удар. Блок. Контратака. Лезвие свистнуло у виска. Перекат влево. Пол скользкий от крови. Он не думал — тело работало само. Ещё один враг. И ещё. Двери в конце коридора. Единственный выход.»*

### 13.5 Учёный

- **Подзаголовок:** Фантастика
- **Описание:**

  > Будущее, которое ещё не наступило — или уже наступило где-то далеко. Космические станции, квантовые компьютеры, параллельные вселенные, искусственный разум. Наука как магия, которую можно объяснить.
  >
  > Язык точный и аналитический. Научная терминология, описания технологий и механизмов. Рассказчик мыслит логически — и ждёт того же от вас.
  >
  > Выборы будут касаться этических дилемм: спасти одного или многих, использовать технологию во благо или для контроля, доверять данным или интуиции.

- **Теги:** `#sci-fi` `#космос` `#технологии` `#исследование` `#будущее` `#наука`
- **Пример:** *«Показания гравитометра не оставляли сомнений: аномалия находилась в 2.7 парсеках. Доктор Чен запустила протокол навигационной коррекции и откинулась в кресле. За иллюминатором звёзды медленно смещались, перерисовывая карту знакомых созвездий.»*

### 13.6 Тёмный маг

- **Подзаголовок:** Мистика
- **Описание:**

  > На границе реального и потустороннего. Руны, ритуалы, тайные знания, сверхъестественные силы. Мир, где каждая тень может оказаться живой, а каждое знание — проклятием.
  >
  > Язык обволакивающий, загадочный. Длинные описания атмосферы, намёки, многозначительные паузы. Рассказчик знает больше, чем говорит — и вы чувствуете это.
  >
  > Ваши выборы будут определять, насколько глубоко вы готовы погрузиться в тьму — и какую цену готовы заплатить за могущество.

- **Теги:** `#мистика` `#тайна` `#магия` `#сверхъестественное` `#ритуалы` `#тьма`
- **Пример:** *«Руны на стене пульсировали слабым сиреневым светом — не приглашая, но и не отвергая. Воздух пах озоном и чем-то древним, чему нет названия на языке смертных. Он знал, что стоит лишь протянуть руку — и портал откроется. Вопрос лишь в том, кто ждёт с той стороны.»*

### 13.7 Детектив

- **Подзаголовок:** Расследование
- **Описание:**

  > Нуар. Дождь. Запертая комната. Труп. И вы — единственный, кто видит то, что другие не замечают. Улики, подозреваемые, ложные следы и одна правда, спрятанная за дюжиной лжи.
  >
  > Язык лаконичный и ироничный. Описания через детали — цвет помады на стакане, царапина на дверной ручке. Внутренний монолог, полный сарказма и наблюдательности.
  >
  > Ваш главный инструмент — дедукция. Собирайте улики, допрашивайте подозреваемых, делайте выводы. Одно неверное обвинение — и убийца уйдёт.

- **Теги:** `#детектив` `#нуар` `#улики` `#дедукция` `#загадка` `#тайна`
- **Пример:** *«Три вещи не сходились: время на часах жертвы, пятно от кофе на подоконнике и показания горничной. Каждая мелочь — как деталь мозаики, которую кто-то старательно перемешал. Я достал блокнот и записал четвёртую вещь: имя человека, который слишком старательно делал вид, что ему всё равно.»*

### 13.8 Хоррор

- **Подзаголовок:** Ужас
- **Описание:**

  > Страх, который подкрадывается медленно. Здесь не будет монстров, выпрыгивающих из-за угла — будет нечто хуже. Сомнение. Ощущение, что реальность не такая, какой кажется. Ненадёжный рассказчик, который, возможно, вам лжёт.
  >
  > Язык тревожный, клаустрофобный. Описания через ощущения: звуки за стеной, запах, которого быть не должно, прикосновение, когда вы одни. Короткие абзацы. Внезапные обрывы.
  >
  > Вы будете принимать решения, не зная, можно ли доверять тому, что вы видите. И каждый выбор может привести вас ближе к правде — или глубже в безумие.

- **Теги:** `#хоррор` `#страх` `#напряжение` `#психология` `#ненадёжный рассказчик` `#безумие`
- **Пример:** *«Дверь была закрыта. Вы точно помните — вы закрыли её минуту назад. Тогда почему сквозняк? И почему на полу — мокрые следы, ведущие от двери к вашему креслу? Вы посмотрели на свои ботинки. Сухие. Вы посмотрели на кресло. На нём кто-то сидел. Только что.»*

### 13.9 Поэт

- **Подзаголовок:** Лирическая проза
- **Описание:**

  > Мир как поэма. Каждое описание — живопись словами. Здесь нет спешки, нет врагов за дверью. Есть дорога, лес, река, небо — и вы, идущий сквозь них, наблюдая и размышляя.
  >
  > Язык метафоричный и ритмичный. Описания природы, красоты в мелочах, философские отступления. Рассказчик не торопит — он приглашает остановиться и увидеть то, что другие пробегают мимо.
  >
  > Выборы здесь не между жизнью и смертью, а между путями — через лес или вдоль реки, заговорить с незнакомцем или пройти мимо. Тихие решения, меняющие всё.

- **Теги:** `#поэзия` `#созерцание` `#природа` `#философия` `#красота` `#тишина`
- **Пример:** *«Река несла в себе отражение облаков — так бережно, словно боялась расплескать. На берегу стоял человек, и тень его, удлиняясь к закату, дотягивалась до самой воды. Куда течёт река? Куда течёт время? Он не знал. Но ему нравилось не знать.»*

### 13.10 Комедиант

- **Подзаголовок:** Юмористическое повествование
- **Описание:**

  > Приключения, от которых болит живот от смеха. Абсурдные ситуации, каламбуры, неожиданные повороты и рассказчик, который не воспринимает себя слишком серьёзно. Четвёртая стена? Какая четвёртая стена?
  >
  > Язык разговорный и ироничный. Неожиданные сравнения, юмористические отступления, комментарии рассказчика к вашим выборам (иногда саркастичные). Если вы выберете нелепый вариант — рассказчик оценит.
  >
  > Идеальный выбор, если вы хотите лёгкую, весёлую историю, не нагружающую драмой. Хотя... кто знает, может, и здесь будет момент, который заставит задуматься.

- **Теги:** `#юмор` `#ирония` `#абсурд` `#приключения` `#лёгкость` `#четвёртая стена`
- **Пример:** *«Дракон посмотрел на рыцаря. Рыцарь посмотрел на дракона. Оба понимали, что битвы не избежать. Оба также понимали, что забыли, зачем, собственно, дерутся. «Может, кофе?» — предложил дракон. Рыцарь задумался. В конце концов, латы у него были чисто декоративные.»*

## 14. Порядок карточек в карусели

Порядок пресетов в карусели — не случайный. Он организован по принципу **контраста** — соседние пресеты максимально отличаются друг от друга, чтобы при прокрутке пользователь ощущал разнообразие:

```
1. Нейтральный (спокойный старт)
2. Рыцарь (эпос)
3. Романтик (чувства — контраст с боевым рыцарем)
4. Боец (динамика — контраст с романтикой)
5. Учёный (интеллект — контраст с физической динамикой)
6. Тёмный маг (мистика — контраст с наукой)
7. Детектив (логика — контраст с иррациональной мистикой)
8. Хоррор (страх — контраст с холодной дедукцией)
9. Поэт (созерцание — контраст с напряжением хоррора)
10. Комедиант (юмор — яркий финал, контраст с тихой поэзией)
```

## 15. Специальные случаи и edge cases

### 15.1 Если пользователь не выбрал пресет и нажал «Продолжить»

Такой ситуации не возникнет, т.к. по умолчанию выбран «Нейтральный рассказчик». Кнопка «Продолжить» всегда активна.

### 15.2 Если пользователь вернулся с игрового экрана (через toolbar → «Меню» → «Новая игра»)

- Экран выбора нарратора — с чистого листа, никакие предыдущие выборы не подсвечены (кроме дефолтного «Нейтрального»).

### 15.3 Быстрый двойной клик по карточке

- Первый клик — выбирает пресет (обновляет описание).
- Второй клик — **не** запускает игру (для этого есть кнопка «Продолжить»). Двойной клик по карточке ≡ одинарному клику. Это защита от случайного запуска.

### 15.4 Resize окна во время просмотра

- Карусель плавно адаптируется: количество видимых карточек пересчитывается.
- Активная карточка остаётся видимой (карусель перецентрируется).
- Debounce на resize: 150ms.
